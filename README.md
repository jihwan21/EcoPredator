# EcoPredator
1. 몬스터 별로 클래스 만들고 고유스킬 창작
2. 메인 코드 완성
3. 강화, 레벨업 시스템 추가

# 클래스
1. 각각의 아이템 클래스화 -> 고유 효과 설정(전투 방어 페이즈에서 사용) -> 상점 클래스 코드 수정
2. 몬스터 각각 클래스화 -> 고유 스킬 설정(전투 공격 페이즈에서 사용)
